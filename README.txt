CRYPTODROP WEB - LISTO PARA SUBIR

OPCION 1: VERCEL (la mas facil)
1) Descomprime este proyecto
2) Sube esta carpeta a un repositorio nuevo en GitHub
3) Entra a vercel.com
4) Login con GitHub
5) Add New Project / Import Git Repository
6) Elige el repo cryptodrop-web
7) Framework: Vite
8) Build Command: npm run build
9) Output Directory: dist
10) Deploy

OPCION 2: LOCAL EN TU PC
1) Instala Node.js LTS
2) Abre la carpeta en terminal
3) npm install
4) npm run dev
5) Abre la URL que aparezca

ARCHIVOS IMPORTANTES
- src/App.jsx = la faucet
- src/main.jsx = arranque React
- package.json = dependencias
- index.html = pagina base
